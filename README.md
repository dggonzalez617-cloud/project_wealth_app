# Project Wealth

A personal financial dashboard built with AI assistance.

**Stack:** Vanilla JS / HTML, Firebase Hosting, GCP Cloud Functions, 
Firestore, Finnhub API, FRED API, Anthropic API (Claude)

**AI Features:**
- Claude-powered market sentiment summary (Economy tab)
- AI-driven IPO intelligence cards with per-company news synthesis
- Built iteratively using Claude as a development partner

**Features:** Live options P&L tracking, net worth across 7 accounts, 
5-year wealth projection, watchlist with SMA signals, macro dashboard, 
trade performance analytics

**Deployment:** Firebase Hosting, access-gated via Google OAuth
